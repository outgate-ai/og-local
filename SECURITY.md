# Security Policy

## Reporting a vulnerability

Please report security issues privately via GitHub Security Advisories:

  https://github.com/outgate-ai/og-local/security/advisories/new

We aim to respond within 3 business days and provide a fix or remediation
plan within 30 days for high/critical issues.

Do **not** open a public issue for security problems.

For other questions, contact support@outgate.ai.

## Supported versions

Until v1.0, only the latest minor release receives security updates.
